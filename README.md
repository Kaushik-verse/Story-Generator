# 🌟 Interactive Story Generator

A simple interactive story game built using **HTML**, **CSS**, and **JavaScript**. Enter your name and make choices to shape your story!

## ✨ Features

- 🔤 Character name input  
- 🌿 Branching story paths based on player choices  
- 🔄 Dynamic text and choice updates  
- 🎨 Clean and minimal UI using standard CSS  
- 🔁 Restart option to replay the story

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)

## 🚀 How to Run

1. **Download or clone** this repository.  
2. Open `index.html` in your browser.  
3. Enter your name and click **"Start Adventure"**.  
4. Make choices and enjoy your custom path through the story!
